# OrangeHRM Login Module Manual Testing Project

## Project Overview

This project demonstrates the complete manual testing lifecycle for the OrangeHRM Login Module.

The objective of this project was to validate login functionality, authentication mechanisms, input validation, password masking behavior, error handling, and navigation features using structured manual testing techniques.
